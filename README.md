# Particle Life

Симуляция искусственной жизни на Rust + Bevy. Несколько типов частиц взаимодействуют по матрице притяжений и отталкиваний — из простых правил возникают кластеры, цепочки и органичные структуры.

## Демо

```
 ● ● ●        ○ ○
●  ●  ●  →→  ○   ○   ←  самоорганизация за ~3 секунды
 ● ● ●        ○ ○
```

## Стек

- **Rust**
- **Bevy** — ECS, рендер, окно
- **bevy_egui** — панель редактора матрицы

## Быстрый старт

```bash
git clone https://github.com/vadych/particle-life
cd particle-life
cargo run --release
```

## Управление

| Клавиша | Действие                                   |
| ------- | ------------------------------------------ |
| `Space` | Пауза / продолжить                         |
| `R`     | Новая случайная матрица                    |
| `T`     | Сброс позиций частиц (матрица сохраняется) |
| `S`     | Показать / скрыть редактор матрицы         |


## Как работает физика

Каждый кадр для каждой пары частиц A и B в радиусе `INTERACTION_RADIUS`:

```
0.0          0.3                1.0   (нормализованное расстояние)
 |────────────|───────────────────|
 ^ отталкивание (жёсткое ядро)   ^ сила из матрицы (+ притяжение, − отталкивание)
```

- **Ближняя зона (0–0.3)** — всегда отталкивание, частицы не проходят сквозь друг друга
- **Дальняя зона (0.3–1.0)** — сила из матрицы, линейно убывает к нулю на границе радиуса
- **Трение** — гасит скорость каждый кадр: `v *= 1 - FRICTION`
- **Тороидальные границы** — частицы появляются с противоположного края

## Матрица взаимодействий

`matrix[A][B]` — сила с которой тип **B действует на тип A**.

```
        0    1    2    3    4    5   ← тип B (источник)
   0  [ ]  [ ]  [ ]  [ ]  [ ]  [ ]
   1  [ ]  [ ]  [ ]  [ ]  [ ]  [ ]  ← тип A (получатель)
   ...
```

Значения от `-1.0` (сильное отталкивание) до `+1.0` (сильное притяжение). Асимметрия — ключ к интересному поведению: A может тянуться к B, пока B убегает от A.

## Параметры (config.rs)

| Константа | По умолчанию | Описание |
|-----------|-------------|----------|
| `NUM_TYPES` | 6 | Количество типов частиц |
| `PARTICLES_PER_TYPE` | 150 | Частиц каждого типа |
| `WORLD_SIZE` | 400.0 | Полуразмер мира (мир от -400 до +400) |
| `PARTICLE_RADIUS` | 3.0 | Визуальный радиус частицы |
| `INTERACTION_RADIUS` | 80.0 | Радиус взаимодействия |
| `FRICTION` | 0.05 | Коэффициент трения |
| `FORCE_SCALE` | 0.5 | Глобальный масштаб силы |
